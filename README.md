
# Java Utilities Collection

This repository contains a set of Java programs designed to perform a variety of utility functions, from arithmetic operations to payroll calculations and time conversions. Each program demonstrates fundamental Java concepts such as user input handling, data processing, and simple calculations.

## Files Included

- **ArithmeticOperations.java**  
  Performs basic arithmetic operations (addition, subtraction, multiplication, division) on user-provided numbers.

- **CharacterAnalyzer.java**  
  Analyzes a given character or string to determine its properties (e.g., letter, digit, uppercase, lowercase).

- **InteractivePayRollSystem.java**  
  A simple payroll system that calculates employee wages based on user input, supporting different payment types.

- **ResBillCalc.java**  
  Calculates restaurant bills, including tips and tax, based on user input.

- **TimeConverter.java**  
  Converts time between seconds, minutes, hours, and days using named constants.

- **tempConvert.java**  
  Converts temperatures between Celsius and Fahrenheit, with user selection through a dialog interface.

- **README.md**  
  This file, providing an overview of the repository and its contents.

## Features

- Demonstrates Java fundamentals: variables, loops, conditionals, and methods.  
- Interactive user input via `Scanner` and `JOptionPane`.  
- Covers practical scenarios: arithmetic, character analysis, payroll, billing, time, and temperature conversion.
